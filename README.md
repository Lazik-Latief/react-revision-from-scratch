# React From Zero 

This repository is created to revise React.js from scratch.
It contains simple notes and examples for understanding core React concepts step by step.
The focus is on building a strong foundation before moving to advanced topics.


## day-17 (basics-notes)

1. React works using components.
2. A component is a JS function that returns JSX.
3. JSX allows writing HTML inside JS.
4. Use {} to write JS inside JSX.
5. JSX must have one parent element.
